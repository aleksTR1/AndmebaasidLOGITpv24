# Kaupade ja Müükide Andmebaas (Sales Database)


See repositoorium sisaldab SQL-skripti andmebaasi struktuuri loomiseks, tabelitevaheliste seoste seadistamiseks ja näidisandmete lisamiseks. Skript on mõeldud käivitamiseks MS SQL Serveris.

## Andmebaasi Struktuur (Skeem)

Projekt koosneb neljast põhilisest tabelist:
1. **Category** – Tootekategooriad (unikaalse nimega).
2. **Customer** – Kliendid koos kontaktandmetega.
3. **Productt** – Tooted, mis on seotud kategooriatega ja omavad hinda.
4. **Sale** – Müügitehingud, mis seovad tooted, kliendid, koguse ja kuupäeva.

---

## SQL Kood / Skript

```sql
-- ========================================================
-- 1. TABELITE LOOMINE JA SEOSTAMINE (DDL)
-- ========================================================

-- Kategooriate tabel
CREATE TABLE Category (
    idCategory INT PRIMARY KEY IDENTITY(1,1),
    Category_name VARCHAR(50) NOT NULL,
    CONSTRAINT UQ_Category_Name UNIQUE (Category_name)
);

-- Klientide tabel
CREATE TABLE Customer (
    idCustomer INT PRIMARY KEY IDENTITY(1,1),
    Name VARCHAR(100) NOT NULL,
    Contact VARCHAR(100)
);

-- Toodete tabel
CREATE TABLE Productt (
    idProduct INT PRIMARY KEY IDENTITY(1,1),
    Name VARCHAR(50) NOT NULL,
    idCategory INT,
    Price MONEY,
    FOREIGN KEY (idCategory) REFERENCES Category(idCategory)
);

-- Müükide tabel
CREATE TABLE Sale (
    idSale INT PRIMARY KEY IDENTITY(1,1),
    idProductt INT,
    idCustomer INT,
    Count INT,
    date_of_sale DATE,
    FOREIGN KEY (idProductt) REFERENCES Productt(idProduct),
    FOREIGN KEY (idCustomer) REFERENCES Customer(idCustomer)
);

-- ========================================================
-- 2. ANDMETE LISAMINE (DML)
-- ========================================================

-- Lisame kategooriad
INSERT INTO Category (Category_name) VALUES ('piima');
INSERT INTO Category (Category_name) VALUES ('autot');

-- Lisame kliendi
INSERT INTO Customer (Name, Contact) VALUES ('vadim', '6767 6767');

-- Lisame tooted (seostame kategooriate ID-dega)
INSERT INTO Productt (Name, idCategory, Price) VALUES ('piim tere', 1, 1.29);
INSERT INTO Productt (Name, idCategory, Price) VALUES ('mercedes', 2, 25000.00);

-- Lisame müügitehingu (toode ID=2, klient ID=1, kogus=10)
INSERT INTO Sale (idProductt, idCustomer, Count, date_of_sale) 
VALUES (2, 1, 10, '2006-10-10');

-- ========================================================
-- 3. ANDMETE KONTROLL (PÄRINGUD)
-- ========================================================

SELECT * FROM Category;
SELECT * FROM Customer;
SELECT * FROM Productt;
SELECT * FROM Sale;
